Como usar o git

Comandos git:

GIT INITI

Inicializa o git no projeto

GIT STATUS

Mostra quais arquivos foram modificados e quais arquivos ainda não foram commitados

GIT ADD + "nome do projeto"

adicionar o arquivo

GIT ADD .

Adiciona todos os arquivo

GIT COMMIT -M + "descriçao do commit"

Upa os arquivos para o git alem de adicionar um breve descrição das alterações

GIT LOG

Mostra os commits e as hash de cada commit

GIT LOG --ALL

Mostra todos os commits e as hash de cada commit

GIT CHECKOUT + "hash do commit"

Avança para o commit especificado

ORGANIZAÇÃO

CLS

Limpa o terminal

HEADER

HEADER siginifica onde em qual commit você esta no momento

MASTER

MASTER é o último commit feito
